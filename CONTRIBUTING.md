# Contributing Guide

## Branch Naming
Branches must follow the pattern:
- `features/<issue-number>`
- `bugs/<issue-number>`
- `issues/<issue-number>`
- `release/<major.minor.patch>`
- `hotfix/<major.minor.patch>`
 
## Merge Strategy
This repository uses **Squash and Merge** only.
- No merge commits allowed
- No rebase merging allowed
